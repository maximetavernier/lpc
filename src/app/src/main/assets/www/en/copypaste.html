How to copy/paste easily !

Avant toute chose, je vous conseille de bien comprendre comment fonctionne le système de fichier, et quel sont les différents types d'entité à l'intérieur (fichier, dossier, raccourcis, etc.)

Faire un copier/coller simple

Dans un premier temps, selectionner le fichier ou le dossier que vous voulez copier. Puis, faites un clique droit sur la souris, un menu contextuel apparait. Cliquez ensuite sur "copier".

Ensuite, il faut coller votre fichier/dossier dans le nouvel emplacement. Allez donc dans le nouvel emplacement, et dans le cadre, faites un nouveau clique droit. Le menu contextuel doit apparaitre légèrement différent. Là, cliquer sur "coller". Attendez de voir le fichier...

Voila !

Faire un copier/coller multiple

Pour faire un copier/coller multiple, c'est la même manipulation, à l'exception près que plusieurs fichiers et/ou dossier sont selectionnés. Pour faire celà, deux méthodes :

  • Selectionner plusieurs fichiers en maintenant le clique droit de la souris
  • Selectionner différents fichiers en maintenant la touche ctrl du clavier
Enfin, comme la méthode simple, il faut d'abord copier les fichiers/dossiers, puis les coller dans la destination. Voila !

Faire un copier/coller avec le clavier

Là encore, ca n'est pas sorcier. Pour faire un copier/coller avec le clavier, il y a 2 combinaison à connaitre.
Pour copier il faut effectuer la combinaison clavier ctrl + c. Si des fichiers sont séléctionnés, alors ces fichiers seront copiés.

Pour ensuite coller les fichiers, il faut, comme vu précédemment, aller dans le dossier de destination, et effectuer la combinaison clavier ctrl + v. Si des fichiers ont été copiés, alors ils seront collés dans le dossier de destination.

Une chose importante à ne pas oublier : quand des fichiers sont copiés, ils restent copiés. Cela veut dire que les fichiers peuvent être collés plusieurs fois.
Une alternative au copier/coller, c'est le couper/coller. Cette alternative permet non pas de seulement copier des fichiers, mais de les déplacer. La combinaison de touche, est ctrl + x pour couper, et ctrl + v pour coller.
