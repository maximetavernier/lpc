Files formats

Recognition and learning of the different kind of file formats

Les formats permettent de reconnaitre les fichiers. Ils sont exclusifs aux fichiers (et non au dossier). Ces formats permettent de définir plusieurs choses : le type de fichier, et la fa√ßon dont il est formaté (d'où l'appelation "format").
Les formats des fichiers sont reconnaissables par son contenu pour l'ordinateur, et par son extension pour l'utilisateur. Toutefois, il faut faire attention ! Un fichier ayant une extension particulière n'est pas forcément un fichier du format associé...

Les formats de documents de bureau

Les formats de documents dits "Office" sont nombreux. Les plus connus ont l'extension utilisés pour la suite Office de Microsoft : doc(x), xls(x), ppt(x)...
Il existe pourtant d'autres type de format. Le plus simple est le texte brut (avec l'extension ".txt") et l'un des plus connus est le type de la société Adobe : ".pdf".
Enfin il existe d'autres types connus, par exemple celui utilisé en web, et interprété par un navigateur : le HTML (avec l'extension ".html", ou ".htm").

Les fichiers audio

Il existe beaucoup de formats audio, mais certains sont plus connus que d'autres. Le MP3 est le plus utilisés (car celui qui prend le moins d'espace mémoire), mais pas celui qui conserve la meilleur qualité.
L'un des meilleurs en qualité à l'heure actuelle est probablement le FLAC (Free Lossless Audio Codec).
