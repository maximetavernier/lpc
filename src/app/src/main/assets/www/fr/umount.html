Comment retirer un appareil USB de manière sauve.

Pour protéger l'intégrité des données qui sont stockés sur un appareil USB, comme une clé ou un disque dur externe, il est conseillé de retirer le périphérique en toute sécurité. Cela signifie interrompre tout protocole de communication entre l'ordinateur et l'appareil USB. Par cette action, cela permet d'assurer que tout les fichiers sont intègres, non seulement sur l'appareil USB, mais aussi du coté de l'ordinateur.

Retirer depuis le bureau

Retirer depuis l'explorateur de fichier

Il existe en effet une autre manière d'éjecter proprement un appareil USB : en passant par le L'explorateur de fichiers. Il suffit d'aller sur l'interface de l'ordinateur afin de trouver les différents disques durs interne ainsi que les appareils USB connectés à l'ordinateur.
Pour enlever un appareil, il suffit de l'identifier et d'effectuer un Clique droit sur l'icone de l'appareil USB, et cliquer sur "Ejecter" dans le menu contextuel :

Voila !

Retirer un appareil USB proprement, cela prend peu de temps, et permet d'assurer que les données sont toujours gardés de manières sures. Bien sur, enlever une clé USB sans passer par cette manipulation ne détruit pas forcément les données. Mais dans le cas où un échange de données se fait quand la clé est enlevé, les données auront la possibilité d'être endommagés...
