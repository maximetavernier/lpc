Formats de fichiers

Reconnaissance et apprentissage des différents formats de fichiers

Les formats permettent de reconnaitre les fichiers. Ils sont exclusifs aux fichiers (et non au dossier). Ces formats permettent de définir plusieurs choses : le type de fichier, et la façon dont il est formaté (d'où l'appelation "format").
Les formats des fichiers sont reconnaissables par son contenu pour l'ordinateur, et par son extension pour l'utilisateur. Toutefois, il faut faire attention ! Un fichier ayant une extension particulière n'est pas forcément un fichier du format associé...

Les formats de documents de bureau

Les formats de documents dits "Office" sont nombreux. Les plus connus ont l'extension utilisés pour la suite Office de Microsoft : doc(x), xls(x), ppt(x)...
Il existe pourtant d'autres type de format. Le plus simple est le texte brut (avec l'extension ".txt") et l'un des plus connus est le type de la société Adobe : ".pdf".
Enfin il existe d'autres types connus, par exemple celui utilisé en web, et interprété par un navigateur : le HTML (avec l'extension ".html", ou ".htm").

Les fichiers audio

Il existe beaucoup de formats audio, mais certains sont plus connus que d'autres. Le MP3 est le plus utilisés (car celui qui prend le moins d'espace mémoire), mais pas celui qui garde la meilleur qualité.
L'un des meilleurs en qualité à l'heure actuelle est probablement le FLAC (Free Lossless Audio Codec).

Les fichiers vidéos

Les formats vidéos sont certes moins nombreux que les formats audio, mais nombreux toutefois. Certains formats de vidéos ont été innovés, ou ont évolués, à l'ère d'Internet, afin de faciliter la lecture rapide et de bonne qualité sur un navigateur web.
Le MP4, le FLV, le WMV, l'AVI, le MKV... ils ont chacuns leurs compressions différentes. Chaque formats sont encodés via des codecs différents, et tous ont leurs spécificités (Prend peu d'espace mémoire, conserve une qualité bonne, voir en Haute Définition, permet l'activation de la 3D, etc.).
Les formats vidéos utilisés sur Youtube, par exemple, sont le MP4 et le FLV, qui sont deux formats très léger mais gardant une qualité d'image très correct.

Les fichiers binaires

Les fichiers dits "Binaires" sont des fichiers pouvant être executés. Ils s'agit tantôt de programme, tantôt de script interprétés par un programme tiers.
Contrairement aux type de fichiers cités précédemment, certains fichiers binaires fonctionnent grâce à d'autres fichiers, que ca soit des fichiers de types médias, ou alors des fichiers de configurations.

Les fichiers de configurations

Les fichiers de configurations sont utiles pour les programmes. Ils permettent soit de se lancer avec une configuration particulières, soit d'effectuer de se comporter différemment, en fonction de ces configurations.
La encore, différents formats existent. Certains sont de simple associations de valeurs (Comme dans les registres de Windows, par exemple), et dont le format le plus connu est "L'ini".
D'autres formats sont plus complexes, ils sont presques des bibliothèques d'information et/ou de configuration. Les deux formats les plus connus sont le XML et le JSON.

Il existe bien d'autres formats de fichiers, la liste est très longue. Par exemple, une archive est considéré comme un type de fichier à part entière, alors qu'il s'agit d'un conteneur possédant une d'autres fichiers qui ont été compressés. Ce type de fichiers est très pratique pour gagner de la place, et donc peut être échangés plus facilement, par email par exemple.
